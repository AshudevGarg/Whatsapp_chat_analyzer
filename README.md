# 📊 WhatsApp Chat Analyzer

🚀 **Live App:**  
[https://your-render-link.onrender.com](https://whatsapp-chat-analyzer-fgfz.onrender.com/)

A powerful WhatsApp Chat Analyzer built using **Python**, **Streamlit**, **Pandas**, **Matplotlib**, and **Seaborn**.

Supports both:

* ✅ Android WhatsApp exports
* ✅ iPhone WhatsApp exports

---

# 🚀 Features

## 📈 Chat Analytics

* Total Messages
* Total Words
* Media Shared
* Links Shared

## 👥 User Analytics

* Most Active Users
* Contribution Percentage
* Person-wise Analysis

## ☁️ Word Analysis

* Word Cloud
* Most Common Words

## 📅 Activity Analysis

* Monthly Timeline
* Weekly Activity
* Hourly Activity
* Heatmaps

## 💬 Chat Extraction

* View chats of specific person
* Download chats as TXT
* Download chats as CSV

## 📱 Cross Platform Support

* Android exported chats
* iPhone exported chats
* Unicode handling
* Multiline messages

# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/AshudevGarg/Whatsapp_chat_analyzer.git
```

Move into project folder:

```bash
cd Whatsapp_chat_analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```

---

# 📥 How to Export WhatsApp Chat

## Android

1. Open WhatsApp
2. Open chat
3. Click 3 dots → More → Export Chat
4. Choose **Without Media**

## iPhone

1. Open WhatsApp
2. Open chat
3. Click Contact/Group Name
4. Export Chat
5. Choose **Without Media**
